# らくつむ用語集

「らくつむ」＝外為どっとコムの外貨積立サービスの愛称。正式商品名は**「らくらくFX積立」**、取引区分は**店頭外国為替証拠金取引**（外貨預金ではない）。

## 1. 基本仕様

| 項目 | 内容 |
|---|---|
| 取扱通貨ペア | 10種類：USD/JPY、AUD/JPY、GBP/JPY、NZD/JPY、CAD/JPY、ZAR/JPY、TRY/JPY、CNH/JPY、MXN/JPY、RUB/JPY（RUB/JPYは2022年3月23日より新規注文受付停止） |
| レバレッジ | 個人：1倍・2倍・3倍から選択（全通貨ペア共通）／法人：1倍のみ |
| レート桁数 | 3桁（例：80.207） |
| 取引単位 | 0.01通貨単位。買付は「通貨単位」または「円貨単位」（円貨は100円以上1円単位、最低1通貨相当） |
| 1回の発注上限 | 全通貨ペア共通で100万通貨 |
| ポジション保有上限 | 全通貨ペア合計で5,000万通貨 |
| 取引手数料 | 無料（コストは買値・売値のスプレッドが実質負担） |
| 税率 | 利益に対して一律20.315%（所得税15%＋復興特別所得税0.315%＋住民税5%） |
| 信託保全 | 会社破たん時も区分管理により全額保全 |
| FXポイント | 各種サービス・キャンペーンで獲得できるポイント。1ポイント＝1円として買付注文に利用可 |

## 2. 証拠金・ロスカット関連

| 用語 | 定義・計算式 |
|---|---|
| **必要保証金** | 買付平均単価 × 取引数量 ÷ レバレッジ で算出。小数点第4位まで計算し、第5位以下は切り上げ |
| **有効比率** | 有効評価額 ÷ 必要保証金額 × 100（%）。10分間隔で判定 |
| **買付停止アラート** | 有効比率が**100%を下回った**時点でメール送信。以後は1分間隔で判定 |
| **ロスカット執行** | 有効比率が**30%を下回った**際に全ポジションを強制決済 |

> 有効比率のライン（100%＝買付停止アラート、30%＝強制ロスカット）は、[rakutsumu_simulator.html](rakutsumu_simulator.html) の「実効有効比率」判定ロジック（`realMarginRatio` / `realMarginClass`）と一致。ただしシミュレーター内の「実効有効比率」「維持率」「実効レバ」「清算価値」は公式用語ではなく、当ブログのシミュレーターが試算のために独自定義した派生指標（[rakutsumu_simulator.html:605-607](rakutsumu_simulator.html#L605-L607)に定義あり）。混同しないよう記事内では区別して表記する。

## 3. 定期買付ルール

| 項目 | 内容 |
|---|---|
| 買付コース | 毎日／毎週（毎週水曜日）／毎月（毎月26日）の3種類 |
| 発注方式 | 成行注文、毎営業日午前11:30から順次執行 |
| 指示期限 | 毎営業日の午前11:30まで |

## 4. スワップポイント

**スワップポイント**＝2か国間の金利差調整額。プラスは受け取り、マイナスは支払い。将来、金利差の縮小・逆転により受取額の縮小や支払いへの転換もあり得る。

受け取り方式は3種類（自動振替・再投資は同時設定不可）：

| 設定 | 定義 | 設定可能金額 | 実行タイミング | 注意点 |
|---|---|---|---|---|
| **貯める** | スワップポイントを日々そのまま蓄積する（ポジションの評価益として保有） | - | - | 自動振替・再投資と異なり口座資金には計上されない |
| **自動振替** | 保有ポジションのスワップポイントが設定金額に達した時点で、らくつむ口座内へ自動で振り替える機能 | 1,000円〜999,999円（1円単位） | 毎営業日午前11:30より順次実行 | 設定金額未満なら実行されず、整数部分のみ振替（小数部分は残存）。再投資との同時設定不可 |
| **再投資** | スワップポイントが設定金額に達した時点で、対象ポジションと同一通貨・同一レバレッジで自動買付する機能 | 100円〜999,999円（1円単位） | 毎営業日午前11:30より順次実行 | 設定金額以上かつ1通貨以上に満たない場合は実行されず、整数部分のみ利用（小数部分は残存）。自動振替との同時設定不可 |

## 5. 取引時間

冬時間（11月第1日曜〜3月第2日曜）・夏時間（3月第2日曜〜11月第1日曜）で切り替わる。

| 区分 | 冬時間 | 夏時間 |
|---|---|---|
| 月曜 | 7:05〜翌6:50 | 7:05〜翌5:50 |
| 火〜金 | 7:15〜翌6:50 | 6:15〜翌5:50 |
| システムメンテナンス（火〜金） | 6:50〜7:15 | 5:50〜6:15 |
| 定期買付執行時刻 | 11:30（共通） | 11:30（共通） |

マイページ：土曜 6:50〜13:00はログイン不可（メンテナンス）。
