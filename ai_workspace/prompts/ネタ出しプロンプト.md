# ブログネタ出しプロンプト

「ブログのネタ出しして」とClaudeに伝えると、以下の手順でネタ候補を提案し、[状況.md](../../状況.md)の「アイデア置き場（Inbox）」に追記する。**自動実行はしない。頼んだ時だけ動く。**

## 手順（Claude向け）

1. `blog/アメブロブログ記事/` 配下の記事（frontmatterの`tags`/`status`/`date`）を読み、既に扱った通貨・テーマ・切り口を把握する
2. Web検索で、rational-labのジャンル（FXスワップ運用、高金利通貨、新NISA、複利モデル）に関連する直近のニュース・金利動向・話題を調べ、上記でまだ扱っていない切り口を探す
3. 3〜5件のネタ候補を挙げる。それぞれ「テーマ／一言理由（なぜ今書く価値があるか）」を明記する
4. 既存記事と重複するテーマは出さない。重複しそうな場合は「差別化できる切り口」を明示する
5. 状況.mdのInboxセクションに以下の形式で追記する：
   ```
   - [YYYY-MM-DD] テーマ名 — 理由（一言）
   ```

## 使用例
- 「ブログのネタ出しして」
- 「トルコリラ以外で高金利通貨のネタを探して」（テーマを絞って依頼することも可能）

## 採用後の運用
候補の中から書くことが決まったら、Inboxの該当行を削除し、[blog_article_prompt.md](blog_article_prompt.md)の手順で執筆に進む。
