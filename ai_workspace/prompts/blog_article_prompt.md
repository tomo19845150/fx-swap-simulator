# ブログ記事生成プロンプト

以下をClaudeに伝えると記事を生成できる。

```
[ブログ名（例：ameba）] に [通貨名/テーマ] についてのブログ記事を書いて。
[blog/<ブログ名>/template.md] と [blog/<ブログ名>/articles/] の過去記事の文体に合わせる。
読者はFX初心者。文字数は[800〜1200字]程度。
```

## 使用例
- 「アメーバブログにメキシコペソ円のスワップ投資について記事を書いて。データは [data/メキシコペソ円データ集.md] を参考に、文体は blog/ameba/template.md に合わせて」

## 記事完成後の運用
1. 完成した記事を `blog/<ブログ名>/articles/` に保存する。
2. 記事が増えたら `blog/<ブログ名>/template.md` を更新依頼する（「テンプレを進化させて」と伝えればよい）。
